# AI Chat App

A simple full-stack AI chat application built with React, Node.js and the OpenAI API.

## Features

- AI-powered chat
- React frontend
- Node.js backend
- OpenAI API integration

## Technologies

- React
- Node.js
- Express
- OpenAI API

## Purpose

This project was created to learn full-stack web development and integrate AI APIs.
